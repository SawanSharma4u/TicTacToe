## Overview

The Tic Tac Toe game in React.

Stack:

- React
- Styles with Sass

## Develop locally

1. Clone this repo by running `git clone https://github.com/SawanSharma4u/TicTacToe.git`
2. Install dependencies by running `npm install`
3. Run `npm run start`
